# BMI Calculator App

A modern and responsive **BMI (Body Mass Index) Calculator App** developed using Flutter.

The application calculates BMI instantly, displays animated results, and provides BMI health information with a clean and smooth user experience.

---

# 🚀 Features

- ✅ BMI Calculation
- ✅ Tween Animation for Result
- ✅ Clear All Values Button
- ✅ BMI Health Information
- ✅ Responsive UI Design
- ✅ Bloc State Management
- ✅ Reusable Widgets
- ✅ Smooth User Experience
- ✅ Clean Architecture Structure

# 🧠 BMI Formula

BMI = Weight (kg) / Height² (m)

---

# 📊 BMI Categories

| BMI Range | Category |
|-----------|-----------|
| Below 18.5 | Underweight |
| 18.5 - 24.9 | Normal Weight |
| 25 - 29.9 | Overweight |
| 30+ | Obesity |
